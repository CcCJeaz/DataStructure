# **栈计算器**
### 功能：
    ​实现四则带任意数量的括号的运算
好像还有一个更难的= = 带三角函数的栈计算器 ( 加入TodoList >_< )
### 实现方法：
    逆波兰表达式求解表达式:
    1.转化为后缀式
    2.利用后缀式求解表达式

### 实践总结：
##### 难点：
    1.多次遇到访问空容器问题
    2.如何妥善的处理 "-" 问题
##### 处理方式：
    1.增加判断空容器条件
    2.这个有不同的解决方案，我在本次实现过程中采用的是将"增零"的办法，即遇到
    "-"时进行在原字符串前加入一个"0"，同时判断是否需在"0"前加上"+"
